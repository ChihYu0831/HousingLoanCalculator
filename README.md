# 個人房貸試算器

## 專案簡介
本程式為個人房貸試算器，使用者可透過簡單輸入快速取得房貸相關資訊。

## 執行說明

### 1️⃣ 輸入項目
使用者需輸入以下資料：
- 房屋總價（元）
- 自備款比例（%）
- 貸款利率（%）
- 貸款年限（年）
- 寬限期（年，選填）

系統提供預設值（如：12000000、20%、2.15%、30年、0年），可直接使用或自行修改。

### 2️⃣ 操作方式
- 開啟程式
- 在「輸入區」填入資料
- 點擊「計算」按鈕
- 顯示計算結果

### 3️⃣ 結果說明
系統輸出以下資訊：
- 貸款總金額
- 每月應繳金額（本息）
- 首期利息
- 首期本金
- 總利息支出
- 總還款金額

計算邏輯包含：
- 寬限期（僅繳利息）
- 本息平均攤還（後續還款）
