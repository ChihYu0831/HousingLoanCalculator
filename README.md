# 個人房貸試算器

## 專案簡介

本程式為個人房貸試算器，使用者透過簡單輸入，快速取得房貸相關資訊。

## 執行說明

1. 輸入項目

使用者需輸入以下資料：
   - 房屋總價（元）
   - 自備款比例（%）
   - 貸款利率（%）
   - 貸款年限（年）
   - 寬限期（年，選填）

系統已提供預設值（例如：12000000、20%、2.15%、30年、0年），可直接使用或自行修改。

2. 操作方式

   - 開啟程式後，在「輸入」填入相關數值
   - 點擊「計算」按鈕
   - 系統即會自動計算並顯示結果

3. 結果說明

系統會顯示以下資訊：
   - 貸款總金額
   - 每月應繳金額（本+息）
   - 首期利息
   - 首期本金
   - 總利息支出
   - 總還款金額

其中計算邏輯包含寬限期（只繳利息）與後續本息攤還，並透過程式自動分段計算
